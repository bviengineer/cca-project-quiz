Question 10

Q). Describe what happens when I request a web page with my browser in terms of computers, networks, files, and object models in the browser. Make and use drawings to help explain!

Quiz Notes for Question 10:

Image obtained from PC Mag Encyclopedia

A). When a user requests a website via their browser, the request is sent via packets first through their network card, then to their internal router and then the ISPs router. The request will follow various routes, from web server to web server until it reaches the destination with the requested files. The information will be returned to the users browser in very much the same manner it was requested, from web server to router, to internet resulting in various hops, to the users ISP, from the ISP router to users router, until the requested files are displayed in their browser.
